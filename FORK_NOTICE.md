# Godelion

This project is a fork and evolution of the **Darwin Gödel Machine (DGM)**,
originally created by Jenny Zhang, Shengran Hu, Cong Lu, Robert Lange, and
Jeff Clune at Sakana AI, as described in:

> **Darwin Gödel Machine: Open-Ended Evolution of Self-Improving Agents**
> Zhang et al., arXiv:2505.22954, 2025
> https://github.com/jennyzzt/dgm

## What Changed

Godelion is a **significant enhancement** of the original DGM with major
improvements in usability, local model support, documentation, safety,
robustness, analysis tooling, and modern Python packaging. See
[CHANGELOG.md](./CHANGELOG.md) for the complete list of changes.
